# Pixiv安卓客户端

- 可以浏览推荐
- 可以查看排行榜
- 可以查看关注的用户
- 可以查看关注的用户的新作品
- 可以查看书签(收藏)
- 查看作品时可以查看相关推荐
- 可以搜索关键字也可以直接搜索ID
- 支持免代理直连(也可以开代理)
- 登录暂时需要代理(VPN)
前往 [Release](https://github.com/xiao-cao-x/pixiv-xiaocao-android/releases) 页面下载
